About Me

Building things,
one line at a time.

I'm Abbas Ibrahim Aliyu — a 200-level Software Engineering student at Northwest University, Kano, with a genuine obsession for how things work on the web. What started as curiosity became something I spend hours on: writing code, breaking things on purpose, and learning from every bug.

I work mainly with HTML, CSS, JavaScript, and Node.js. I'm not just learning these things in a classroom — I'm actually using them. My portfolio is the first project I built entirely from scratch, and it's one I'm proud of.

I completed the freeCodeCamp Responsive Web Design program, which pushed me to think about design and accessibility in ways that classes alone don't always cover. When I'm not coding, I'm probably reading about something tech-related or figuring out what to build next.

How I Think

Values & Approach

How I work matters as much as what I build. These are the things I keep coming back to, no matter what I'm working on.

✦

Curiosity First

I ask "why" before "how". Understanding the reason behind something makes the how much cleaner. I'd rather spend an extra hour understanding a concept than copy-paste a solution I don't get.

◈

Ship, Then Improve

Waiting for perfect is the enemy of done. I'd rather build something that works, put it out there, and iterate — than sit on an idea forever. My portfolio is proof of that.

⬡

Clean & Readable

Code is read more than it's written. I care about naming things properly, keeping functions focused, and writing code that a future version of me — or someone else — can actually follow.

△

Always Learning

The tech world moves fast, and I like that about it. freeCodeCamp, documentation, YouTube, GitHub repos of people I admire — I'm always picking something up from somewhere.
